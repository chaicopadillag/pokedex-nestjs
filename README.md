## Stack Usado

- Node.js
- NestJS
- MongoDB

## Requisitos

```
npm i -g @nestjs/cli
```

## Instalacion

```
 yarn
```

## Crear en archivo de configuracion **.env**

```
 cp .env.example .env
```

## Docker Base de Datos

```
 docker-compose up -d
```

## Correr el Servidor

```
 yarn dev
```
